# iOS Screenshot Resize & App Store Upload Guide

## Overview

This guide walks you through resizing the 5 generated QwikShift app screenshots to the 3 required iOS device formats, then uploading them to App Store Connect.

**Assets Generated:** 5 unique screenshots + 1 app icon  
**Total Files Needed:** 15 screenshots (5 designs × 3 device sizes) + 1 app icon  
**Time Required:** ~30-45 minutes

---

## Part 1: Screenshot Resizing

### Required Dimensions

You need to create 3 versions of each screenshot at these exact pixel dimensions:

| Device | Width | Height | Aspect Ratio |
|--------|-------|--------|--------------|
| iPhone 6.7" (Pro Max) | 2796 | 1290 | 2.17:1 |
| iPhone 6.1" (Standard) | 2556 | 1179 | 2.17:1 |
| iPad 12.9" | 2048 | 2732 | 0.75:1 |

### Screenshots to Resize

1. **Screenshot 1:** Find Event Jobs Near You (map view)
2. 2. **Screenshot 2:** Review Job & Apply in Minutes (job details)
   3. 3. **Screenshot 3:** Verified & Rated Workers (worker profile)
      4. 4. **Screenshot 4:** Secure Payment Processing (payment confirmation)
         5. 5. **Screenshot 5:** Protected by QwikShift Guarantee (protection features)

                2.    2. **Select Your Screenshots:**

                                                                                                         ## Option 2: Using ImageMagick (Command Line)
                                                                                                         
                                                                                                         If you're comfortable with command line, this is faster for batch operations:
                                                                                                         
                                                                                                         ```bash
                                                                                                         # Navigate to your screenshots folder
                                                                                                         cd ~/Downloads

                                                                                                         # Resize for iPhone 6.7" (2796x1290)
                                                                                                         for f in screenshot-*.png; do
                                                                                                           convert "$f" -resize 2796x1290 -background white -gravity center -extent 2796x1290 "iPhone67-${f%.*}-resized.png"
                                                                                                         done

                                                                                                         # Resize for iPhone 6.1" (2556x1179)
                                                                                                         for f in screenshot-*.png; do
                                                                                                           convert "$f" -resize 2556x1179 -background white -gravity center -extent 2556x1179 "iPhone61-${f%.*}-resized.png"
                                                                                                         done

                                                                                                         # Resize for iPad 12.9" (2048x2732)
                                                                                                         for f in screenshot-*.png; do
                                                                                                           convert "$f" -resize 2048x2732 -background white -gravity center -extent 2048x2732 "iPad129-${f%.*}-resized.png"
                                                                                                         done
                                                                                                         ```
                                                                                                         
                                                                                                         Install ImageMagick if needed:
                                                                                                         - **macOS:** `brew install imagemagick`
                                                                                                         - - **Windows:** Download from https://imagemagick.org/script/download.php
                                                                                                           - - **Linux:** `sudo apt-get install imagemagick`

                                                                                                                                                                                                                                                                                                     7. ## Troubleshooting
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                     8. ### Issue: Screenshot dimensions rejected
                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                     9. **Solution:** Ensure exact pixel dimensions. Use the verification command:
                                                                                                                                                                                                                                                                                                     10. ```bash
                                                                                                                                                                                                                                                                                                         # macOS/Linux
                                                                                                                                                                                                                                                                                                         identify your-screenshot.png
                                                                                                                                                                                                                                                                                                         # Should output: 2796x1290 (or 2556x1179, or 2048x2732)
                                                                                                                                                                                                                                                                                                         ```
                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                         ### Issue: App icon rejected (wrong size)
                                                                                                                                                                                                                                                                                                         
                                                                                                                                                                                                                                                                                                         **Solution:** Icon must be exactly 1024×1024 PNG with transparent background. Regenerate if needed:
                                                                                                                                                                                                                                                                                                         - Use Photopea: Image → Canvas Size → 1024×1024
                                                                                                                                                                                                                                                                                                         - - Or use TinyPNG + Photoshop/GIMP
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                           - ### Issue: Screenshots look distorted/stretched
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                           - **Solution:** Make sure you use "contain" or "fit" mode when resizing, not "stretch" mode. This adds padding rather than distorting the image.
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                           - ### Issue: Can't find App Store Connect section
                                                                                                                                                                                                                                                                                                          
                                                                                                                                                                                                                                                                                                           - **Solution:** Navigate directly to:
                                                                                                                                                                                                                                                                                                           - 1. https://appstoreconnect.apple.com
                                                                                                                                                                                                                                                                                                             2. 2. Apps → QwikShift → Version (e.g., "1.0")
                                                                                                                                                                                                                                                                                                                3. 3. Left sidebar → "Screenshots and Preview"
