                                                                                                        ## Resizing Command (ImageMagick)
                                                                                                        
                                                                                                        ```bash
                                                                                                        # iPhone 6.7"
                                                                                                        convert original.png -resize 2796x1290 -background white -gravity center -extent 2796x1290 screenshot-iphone-67.png

                                                                                                        # iPhone 6.1"
                                                                                                        convert original.png -resize 2556x1179 -background white -gravity center -extent 2556x1179 screenshot-iphone-61.png

                                                                                                        # iPad 12.9"
                                                                                                        convert original.png -resize 2048x2732 -background white -gravity center -extent 2048x2732 screenshot-ipad-129.png
                                                                                                        ```
                                                                                                        
                                                                                                        ---
                                                                                                        
                                                                                                        ## Batch Resizing Script
                                                                                                        
                                                                                                        ```bash
                                                                                                        #!/bin/bash
                                                                                                        # Save as: resize-screenshots.sh
                                                                                                        # Run from ios/app-store-assets/screenshots/ directory

                                                                                                        for file in *.png; do
                                                                                                            # Get filename without extension
                                                                                                            filename="${file%.*}"

                                                                                                            # iPhone 6.7"
                                                                                                            convert "$file" -resize 2796x1290 -background white -gravity center -extent 2796x1290 "$filename-iPhone-67.png"

                                                                                                            # iPhone 6.1"
                                                                                                            convert "$file" -resize 2556x1179 -background white -gravity center -extent 2556x1179 "$filename-iPhone-61.png"

                                                                                                            # iPad 12.9"
                                                                                                            convert "$file" -resize 2048x2732 -background white -gravity center -extent 2048x2732 "$filename-iPad-129.png"
                                                                                                        done

                                                                                                        echo "All screenshots resized successfully!"
                                                                                                        ```
